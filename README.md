# LinkedIn-Scraper
Simple Selenium Scraper to Verify Recruiters data &amp; their LinkedIn from databases.

------ WORK IN PROGRESS --------

## Setup

Run 

`pipenv shell`

Install 

`pipenv install`

Run the scraper by

`python3 main.py`
